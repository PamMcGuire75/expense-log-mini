# Expense Log Mini

A simple browser-based expense tracker (Canada-focused) with:

- GST auto-calculation
- Net (before GST) + “Total includes GST” toggle
- Category summaries
- CSV export
- Local storage persistence

## Live Demo
https://pammcguire75.github.io/expense-log-mini/

## Features
- Real-time GST + Net calculations
- Reverse GST logic
- Category breakdown totals
- Works entirely in the browser

Built with vanilla HTML, CSS, and JavaScript.
